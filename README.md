# Моделирование вероятностных экспериментов

## Описание проекта
Этот проект содержит реализации различных вероятностных экспериментов:
1. **Геометрическая вероятность** - моделирование попадания в сложные фигуры
2. **Частотная вероятность** - статистические эксперименты с большим числом испытаний
